    Sofia Design Tokens

    The design tokens defined in sofia are delivered using SASS modules. This overview describes all the modules defined in this package.

    In order to use the different modules, the module must be imported into the scss-file with the @use rule.

    Caution: if the token is represented by a variable, the variable name starts with a $, if the token is represented by a mixin, the variable name does not have a $ in the name.

    Examples

    Sofia Colors (Variables):

    @use "~@celosnext/material-theme/material-theme/modules/sofia-color";

color: sofia-color.$blue-50;

    Sofia Spacings (Variables):

    @use "~@celosnext/material-theme/material-theme/modules/sofia-spacing";

margin: sofia-spacing.$scale-8 sofia-spacing.$scale-16;

    Sofia Shadows (Variables):

    @use "~@celosnext/material-theme/material-theme/modules/sofia-shadow";

box-shadow: sofia-shadow.$inner-blur-4

    Sofia Typography (Mixins):

    @use "~@celosnext/material-theme/material-theme/modules/sofia-typo";

.headline-primary {
  @include sofia-typo.heading-primary
}
