---
title: 工場出荷状態への初期化
---

IoTconnector flexを工場出荷状態の設定に戻すには、「USER」ボタンを5秒間以上長押ししてください。

![IoTconnector flexユーザーボタン](/img/IoT2050UserButton.png)

工場出荷状態にリセットされる項目:

- ログインパスワード
- その他すべての機器設定

工場出荷状態にリセットされない項目:

- ログファイル

リセット後、初期パスワードでログインすると (参照: [`Login`](GettingStarted.md#login)) 新しいパスワードの設定と設定ウィザードの再実行が要求されます。
